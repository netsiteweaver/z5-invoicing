1. Overview

The Z5 Invoicing System manages products, customers, orders, sales, inventory, suppliers, and user access across departments/locations. It provides dashboards, role-based permissions, and printable documents.

2. Access & Accounts

  • URL: Provided by your administrator
  • Browsers: Latest Chrome, Firefox
  • Devices: Laptops, Tablets. Smartphones are not fully supported.
  • Login: Enter email/username and password; optionally enable Remember Me
  • Forgot password: Use “Forgot your password?” to get a reset link
  • Sign out: Profile menu → Logout
  • Account issues: Contact an administrator

3. Navigation & UI Basics

  • Main navigation: Left sidebar modules (Dashboard, Customers, Products, Orders, Sales, Inventory, Suppliers, Settings, Users)
  • Search and filters: List pages include search and filter controls
  • Pagination: Controls at the bottom of list pages
  • Common actions: View, Edit, Delete/Deactivate, Print/Export

4. Roles & Permissions

  • Normal: Access to assigned features
  • Admin: Broad access including configuration and user management

If a menu or button is missing, your role may not allow it.

5. Customers

What you can do

  • Create, edit, and deactivate customers; search and filter the list
  • View basic customer details on the list and a full detail page

Create a customer

  1. Go to Customers → Add Customer
  2. Enter company and contact details
  3. Save

Find and manage customers

  • Use the search bar (name, email, phone) and status filters
  • Open a customer to view details and edit

6. Suppliers

What you can do

  • Create, edit, and soft-delete suppliers (inactive suppliers are hidden by default)
  • Search by name, contact, phone, email; filter by status

7. Products (Catalog)

What you can do

  • Create and edit products with pricing
  • Manage categories and brands

8. Orders

Create an order

  1. Orders → New
  2. Select a customer and add items. Order Status is set to draft by default, but can be changed to pending or confirmed.
  3. Save

Manage

  • Edit while allowed.
  • When the order is confirmed, it can be converted to a sale.

9. Sales

Create a sale

  • Direct: Sales → New
  • From Order: Orders → open order → Convert to Sale

10. Inventory

What you can do

  • View stock by product and department
  • Record stock movements (in/out/transfer)
  • Low stock view and stock reports

11. Goods Receipts

  • Record received goods with multiple items, costs and VAT breakdown
  • Approve receipts to update inventory; print GRN as PDF

12. Stock Transfers

  • Create transfers between departments; receive transfers; print documents

13. Profile & Password

  • Profile: Update name/email via Profile page (top-left avatar)
  • Password: Update password on the Profile page

14. Settings (Admin)

  • Company settings and general configuration

15. User Management (Admin)

  • Manage users, roles, and permissions

16. Documents & Printing

  • Many details pages provide printable/PDF documents (e.g., Goods Receipts, Stock Transfers)

17. Tips & Troubleshooting

  • Can’t log in: Use Forgot Password; if still blocked, contact admin
  • Missing options: Your role might not allow it
  • PDF blocked: Allow popups for the site
